# Simple Custom Post Plugin

A beginner-friendly WordPress plugin that registers a custom post type
called **Projects** using WordPress hooks.

## Features
- Custom post type registration
- Uses add_action and register_post_type
- Clean and minimal PHP code
- Secure WordPress plugin structure

## Tech Stack
- WordPress
- PHP
- Hooks & Actions

## Purpose
This project was created to understand WordPress plugin development
and core WordPress architecture.
